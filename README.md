# iMedic
iMedic Agent is an advanced AI-powered healthcare assistant built using Google AI Agent Development Kit (ADK). It is designed to streamline and automate key patient-care service workflows with high accuracy, fast responses, and seamless integration with medical systems.

The agent intelligently supports multiple healthcare operations, making it a unified digital companion for both patients and clinics.

#Medical Information Assistance
Basic health-related queries (non-diagnostic)
Medication guidelines and general safety information
It serves as a first layer of intelligent support for patients seeking quick medical information.

#Doctor Appointment Booking
The agent can schedule doctor appointments in real time by:
Checking doctor availability
Suggesting next available slots
Confirming bookings
Updating or canceling appointments
It integrates easily with custom tools.

#Online Medicine Order & Confirmation
iMedic can manage the full medicine ordering workflow
Checking pharmacy inventory
Processing user medicine lists
Calculating total price with delivery/service charges
Confirming and placing the order

#Medical Pre-Authorization Support
The agent can assist patients with pre-authorization requirements by:
Collecting patient and treatment details
Verifying policy information
Checking insurer requirements
Preparing a pre-authorization request summary
Guiding the user on supporting documents

#Built with Google ADK
iMedic Agent is created using Google AI Agent Development Kit, which enables:
Function calling with structured responses
